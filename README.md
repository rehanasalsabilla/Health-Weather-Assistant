# 🌦 Intelligent Weather-Based Health Recommendation System using AI

**Web Title:** Health Weather Assistant  

An AI-powered web application that analyzes real-time weather and air quality data to provide personalized health recommendations and environmental risk insights.

This system helps users understand how current environmental conditions may impact their health and daily activities.

---

## 🚀 Overview

The **Health Weather Assistant** is an intelligent decision-support system that integrates:

- Real-time Weather Data
- Air Quality Monitoring
- AI-based Health Risk Analysis
- Personalized Health Recommendations

The application uses Artificial Intelligence to interpret environmental conditions and generate human-friendly insights such as:

- Health Risk Level
- Comfort Index
- Outdoor Activity Recommendation
- Sensitive Group Warning
- Daily Health Advice

This project simulates a **smart environmental health assistant** that can be used in smart city, public health, or personal wellness scenarios.

## 🧩 System Architecture

User → React Frontend → FastAPI Backend → External APIs + AI Model → Database

### Data Flow

1. User enters a city name
2. Backend fetches weather data from Weather API
3. Backend fetches air quality data
4. AI model analyzes environmental condition
5. AI generates structured health recommendation
6. Result displayed in dashboard UI
7. History stored in SQLite database
